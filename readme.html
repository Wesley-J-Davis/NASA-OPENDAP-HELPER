OpenDAP Server Download Tips

Step 1 - Access Data Server

OpenDAP data server link: https://opendap.nccs.nasa.gov/dods/GEOS-5/fp/0.25_deg

Step 2 - fcast / assim / seamless

fcast

GEOS forecasts - two variants:

  • LATEST
    Example: https://opendap.nccs.nasa.gov/dods/GEOS-5/fp/0.25_deg/inst1_2d_hwl_Nx.latest
    Time variable starts 10 days prior to current time, leading up to current time.

  • HISTORICAL
    Example: https://opendap.nccs.nasa.gov/dods/GEOS-5/fp/0.25_deg/fcast/inst1_2d_hwl_Nx.20260414_00
    Time variable starts at the labeled assimilation time and goes for 10 days.

assim

All GEOS assimilations from 01 Dec 2017 to most recent.

Example: https://opendap.nccs.nasa.gov/dods/GEOS-5/fp/0.25_deg/assim/tavg3_3d_udt_Nv
Time variable starts at 01 Dec 2017 and goes up to the most recent assimilation time.

seamless

Assimilations up to a date and then 10 days of forecast data for that assimilation date.

  • LATEST
    Example: https://opendap.nccs.nasa.gov/dods/GEOS-5/fp/0.25_deg/inst1_2d_hwl_Nx.latest
    Time variable starts 10 days prior to current time, leading up to current time.

  • HISTORICAL

    Example: https://opendap.nccs.nasa.gov/dods/GEOS-5/fp/0.25_deg/assim/tavg3_3d_udt_Nv.20260328_12
    Time variable starts at 01 Dec 2017 and is assimilations up to the labeled date, plus 10 days of forecast data beyond that.

Step 3 - Choose Collection to Download

  1. inst1_2d_hwl_Nx - 2d, 1-Hourly, Instantaneous, Single-Level, Forecast, Hyperwall
  2. inst1_2d_lfo_Nx - GEOS5 FP 2d time-averaged land surface forcing
  3. inst1_2d_smp_Nx - GEOS5 FP 2d instantaneous diagnostics for SMAP
  4. etc...

Step 4 - Choose Whether to Download Entire Dataset or Just a Subset

Entire Dataset

Using Python:

import xarray as xr
from pathlib import Path
import sys

output_dir = Path('downloads')
output_dir.mkdir(exist_ok=True)

urls = ["https://opendap.nccs.nasa.gov/dods/GEOS-5/fp/0.25_deg/fcast/inst3_3d_aer_Np.latest"]

print(f'Downloading {len(urls)} files...')

for i, url in enumerate(urls, 1):
    filename = output_dir / f'geos_fp_subset_{i:04d}.nc'
    print(f'[{i}/{len(urls)}] Downloading {filename.name}...')
    try:
        ds = xr.open_dataset(url)
        ds.load().to_netcdf(filename)
        print(f'  ✓ Success')
    except Exception as e:
        print(f'  ✗ Failed: {e}')

Subsetting the Dataset with Python

The code above remains the same, but the URL will depend on your subsetting constraints. The format is as follows:

${opendap_url}?var[times][lats][lons][levels],time[times],lat[lats],lon[lons],level[levels]
${opendap_url}?var[times][lats][lons],time[times],lat[lats],lon[lons]

Example of subsetting by time/lat/lon while keeping all levels for multiple variables:

https://opendap.nccs.nasa.gov/dods/GEOS-5/fp/0.25_deg/fcast/inst3_3d_aer_Np.latest?lwi[57:80][0:720][0:1151],ps[57:80][500:720][800:1151],time[57:80],lat[500:720],lon[800:1151]
  • If a dimension is left out of the call, it implicitly means "include all levels" in the download.
  • Desired variables from the collection are explicitly defined along with their subset.
  • At the end of the URL, the overall dimensions to be subsetted must be explicitly defined.
